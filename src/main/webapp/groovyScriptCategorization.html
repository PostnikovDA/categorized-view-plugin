Specify a groovy script that can create arbitrary categorizations based on jobs

The groovy script has a single input: a variable named "jobToCategorize" which represents the job that requires categorization. The script must return either:

  • null if the job can't be categorized at all
  • any string representing the name of the category in which the job fits.
Example 
if (jobToCategorize.name.contains('apple') || jobToCategorize.name.contains('banana'))
	return 'Fruits'

return null
